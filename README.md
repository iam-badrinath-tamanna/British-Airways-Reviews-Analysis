# British-Airways-Reviews-Analysis
Tableau dashboard analyzing British Airways customer feedback and satisfaction trends.


## Project Overview
This project features an interactive data visualization dashboard built to analyze British Airways customer reviews from March 2016 to October 2023. The objective is to extract actionable business intelligence from passenger feedback and identify key trends in customer satisfaction.


## Key Insights & Features
* **Core KPIs:** Tracks average scores for overall rating, cabin staff service, food & beverages, entertainment, and seat comfort.
* **Time-Series Analysis:** Visualizes fluctuating passenger satisfaction trends over a 7-year period to identify performance peaks and drops.
* **Geospatial Mapping:** Highlights average overall ratings by the reviewer's country to uncover regional variations in customer sentiment.
* **Aircraft Segmentation:** Analyzed the correlation between specific aircraft models (e.g., Boeing 747-400 vs. A320) and customer satisfaction levels, alongside total review volume.

## Tools Used
* **Data Visualization:** Tableau
* **Data Processing & Analytics:** Excel 

## How to View the Project
1. Download the `British_Airways_Review_Dashboard.twbx` file from this repository.
2. Open the file using [Tableau Desktop](https://www.tableau.com/products/desktop) or the free [Tableau Reader](https://www.tableau.com/products/reader).
